Conversational Recommender System – Validation Dataset

This repository contains the validation dataset used to evaluate the Hybrid Conversational Recommender System (H-CRS) against a baseline ChatGPT model.
The dataset includes six conversational topics designed to test accuracy, relevance, and factual grounding in an e-commerce context.

Dataset Overview

Each topic represents a distinct interaction type:

General Questions – Open-ended brewing questions.

Avoid Not-Related Subjects – Off-domain prompts for intent filtering.

Support for Specific Products – Catalog-linked factual queries.

Product Recommendation – Grounded product suggestions.

Product Comparisons – Comparative reasoning between items.

Boundary and Clarification Handling – Ambiguous queries requiring clarification.

Each entry includes:

The user question,

The H-CRS response,

The ChatGPT baseline response,

And optional notes on expected behavior.
